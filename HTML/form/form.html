input 태그 attribute 사용

  • placeholder (text가 아님!!가상안내문구일뿐 value가 아님)
  • minlength
  • maxlength
  • required : 해당 input에 값을 입력 받을때까지 안내문을 보여줌
  • disabled : 값을 입력하지 못하도록 input 비활성화
  • value : input 초기값(value) !!

input type 사용하기

  • email : @ 가 없으면 error 반환
  • password
  • url : url주소가 아니면 error 반환
  • number : attribute 'min', 'max' 활용 가능 number가 아니거나
    min,max 조건에 맞지 않으면 error 반환
  • file : attribute 'accept' 활용하여 허용하는 파일형식 지정 가능
